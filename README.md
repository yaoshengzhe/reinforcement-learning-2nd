# Reinforcement Learning: An Introduction (2nd Edition)

Exercises, code implementations, and notes from reading *Reinforcement Learning: An Introduction* by Richard S. Sutton and Andrew G. Barto (2nd Edition).

## About the Book

This is a companion repository for the classic textbook on reinforcement learning. The book is freely available at:
http://incompleteideas.net/book/the-book-2nd.html
